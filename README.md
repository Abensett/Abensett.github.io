# Portfolio — Aimen

Personal portfolio. Vanilla HTML/CSS, no framework.

## Stack

- HTML5 + CSS3 (custom properties, 3D transforms, animations)
- Zero JS frameworks, zero build step
- Fonts: Poppins via Google Fonts

## Features

- Dark mode with localStorage persistence
- CSS 3D geometric shapes (pyramid, cube, ring, diamond, triangle), 9 per section
- Glow effects in dark mode without breaking `preserve-3d`
- Responsive: 1024 / 860 / 640 / 400px breakpoints
- Animated logo (GIF, plays once, transparent background)
- Project cards with stack tags and concept keywords

## Structure

```
index.html       main page
css/style.css    all styles
img/             images, logo, favicons
docs/            technical notes
```

## Sections

- Hero
- About + skills
- Experience (timeline)
- Projects (featured Webserv + 12 cards)
- Contact
