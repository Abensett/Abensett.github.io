**Objectif:** Reconstruire le portfolio depuis le squelette existant avec un vrai contenu et une identité visuelle cohérente.

**Done:**
- Analysé l'état du repo (pack index corrompu, rebuilt avec `git index-pack`)
- Reécrit `index.html` et `css/style.css` from scratch
- Contenu réel : bio, projets avec résumés (Webserv featured, Inception, ft_transcendence, Minishell, Security track, EssenceMoinsCher, Security Toolkit)

- Ajouté 3 nouvelles formes géométriques CSS 3D : cube wireframe (6 faces `preserve-3d`), ring, diamond, triangle plat
- 9 formes décoratives par section, tailles et vitesses variées
- Dark mode complet : `body.dark`, variables CSS, `localStorage`, transitions 350ms
- Glow en dark mode : `box-shadow` sur faces cube, `filter: drop-shadow` sur pyramides/triangles, `box-shadow` double couche sur ring/diamond
- Bouton dark mode dans la nav : pill avec icône ☽/☀
- Responsive mobile : breakpoints 860/640/400px, formes décoratives cachées sur mobile, `--s` pour redimensionner la pyramide

**Learned:**
- `filter` sur un parent de `preserve-3d` aplatit le rendu 3D — appliquer le glow sur les enfants (`.face`) pour les cubes
- CSS custom property sur un élément = scope local, suffisant pour varier la taille sans nouvelle classe
- `animation-direction: reverse` sur même keyframe = rotation opposée, zero code dupliqué
